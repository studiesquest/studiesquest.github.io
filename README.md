## 🕹️ Playability Ratings

Since the sign-in page doesn't support cursor lock, games are rated:

| Badge | Meaning |
|-------|---------|
| 🟢 **Playable** | Works perfectly — keyboard only or click-based |
| 🟡 **Slightly Unplayable** | Mostly works, some mouse movement needed |
| 🔴 **Hardly Playable** | Needs mouse aiming / FPS controls — cursor will escape the window |
